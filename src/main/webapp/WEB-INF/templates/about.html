Sobre esse Servidor

Esse servidor faz parte do projeto integrado SDC-X (Screening for Diabetes Complications) do Instituto de Engenharia Biomédica (IEB) da Universidade Federal de Santa Catarina (UFSC).

This server is built from a number of modules of the HAPI FHIR project, which is a 100% open-source (Apache 2.0 Licensed) Java based implementation of the FHIR specification.

Dados no servidor

Esse servidor possui dados de pacientes reais e pode ser acessado apenas por pessoal autorizado.

Este exemplo demonstra as partes importantes de um resource: uma extensão local, a apresentação HTML legível e o conteúdo de dados definido padrão.
